## API

可直接到程式檔案內看說明，或 `clone` 專案後下 `make docs` 指令產生 markdown 文件。

## 開發

### 環境

* iojs v1 v2
* git
* aws command line tool

### 取得原始碼

```bash
$ git clone git@github.com:YouMeb/analytics.js.git
```

### 安裝專案必要 Node.js 工具

```bash
$ npm i
```

### 建立 standalone 檔案

```bash
$ make
```

### 發布新版本

```bash
$ make release
```
